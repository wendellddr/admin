# admin
Project Admin
=======

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the migrations

```bash
# generate migration automatically
$ yarn typeorm migration:generate -n Example

# generate migration blank
$ yarn typeorm migration:create -n Example

# run migration
$ yarn typeorm migration:run
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
